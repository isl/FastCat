HOW TO USE THE LOCATION SECTION

Inside this section you will find all locations used in the records that you have published to Fast Cat team. Here you can add information for the identity of locations that you would like to appear on maps in the Research space.

To do that follow these steps:

1.      Go to the gear button on the right side of the location and right click “edit location”

2.      Press the button: “Ask Getty”

2.1  If Getty gives you results choose the correct one and choose save

2.2  If you get no results from the “Ask Getty” button, you can click on the link: “Getty location database” and try to find the location there. If you find the location in Getty database copy paste the TGN Id in the corresponding field.

2.3  If you don’t find the location in the Getty database you can go to google maps, find the coordinates of the location and copy paste the coordinates in the corresponding field. You can also correct the location name or translate it in the “Corrected/vernacular location name”. Else we will keep the location name of the source.

3.      If you want to add another extra location name you can add this in “Other Location name(optional)”

Field description:

Source Location name: The Location name as it was written in the source (This will be the main location name , except if you add a “corrected/vernacular Location name” which will replace the source location name in the Research space –not in Fast Cat record.)

Corrected/ Vernacular Location name: If the location in the source has spelling mistakes or illegible content or is written in non latin alphabet you can add here the corrected vernacular location name* or the English location name. This  will replace the Source location name in the Research space–not in Fast Cat record)

Other Location name: If you have already added a corrected vernacular, or you want to keep the source location name but you want to add another denomination of the Location (e.g. an historical name), then you add it here.

Location Type: The type of the Location (e.g. city, province etc.). This you can add only inside the Fast Cat record in parenthesis next to the location name. If you want to change this you have to go to the Fast Cat records it was used and change it from the “Edit Vocabulary” functionality.

Broader Location name: A broader location name (e.g. Athens, Greece). This you can add only inside the Fast Cat record with slash (/) next to the location name. If you want to change this you have to go to the Fast Cat records it was used and change it from the “Edit Vocabulary” functionality.

TGN ID: This is the TGN ID of the specific Location. This can be found automatically by pressing the “Ask Getty” button or you can go to “ Getty location database” and copy paste it here.

Coordinates: The coordinates of the locations (Decimal Degrees). This can be found automatically by pressing the “Ask Getty” button or you can go to Google maps, right click on the location and copy paste here the coordinates.

Additional information:

·         The position info is dark orange when the location is used in more than one different positions.

·         If one of the records that includes this location has been published to the Research space the status next to the location will change to “published”.

 *Vernacular location name: The location name, as it is referred to in its county, but written in Latin alphabet.
